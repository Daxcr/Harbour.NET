# Harbour.NET

A basic C# Discord bot which lets you deploy other C# Discord bots.

## Commands

- **/mount**: This command sets up a new bot.
- **/token**: Generates a key which lets you upload a token via the bot's frontend.
- **/pull**: Pull a bot's source.
- **/compile**: Compile the bot's source.
- **/stage**: Replace the bot's current assembly with the newly compiled assembly.
- **/deploy**: Deploys a bot.
- **/stop**: Stops a bot.
